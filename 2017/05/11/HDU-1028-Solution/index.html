发表于 2017-05-11 | | 阅读次数

题意

给你一个大于0的数n， 把它拆成一系列正整数的和，有多少种拆法？

解法

组合数可以用多项式系数来表示，比如：

右式$k$次项的系数表示了用最多2个1，最多2个2，最多2个3能拼出$k$的方案数（手动计算，不合并同类项就看出来了），于是在本题中要计算的就是

的展开式的n次项的系数，因为n是有限的，（）中的多项式的项的次数是非负的，所以在计算的时候截取（）中所有次数小于等于$n$的项即可

代码
